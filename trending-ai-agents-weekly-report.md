## Key Trends This Week

1. **Self-Evolving Agents** — 3 of the top 15 repos (hermes-agent, GenericAgent, evolver) focus on agents that evolve/grow their own capabilities. This is the hottest paradigm shift.

2. **Agent Behavior Control** — Massive demand for config files (CLAUDE.md, AGENTS.md) that make agents behave properly. Karpathy-skills hit #1 on GitHub trending with +40K stars in a week.

3. **Persistent Memory** — claude-mem (+10K weekly) shows the critical need for agents that remember across sessions.

4. **Multi-Agent Platforms** — multica, swarm-forge, CrabTrap all address multi-agent coordination and production security.

5. **Design + Agents** — huashu-design and cc-design show agents are expanding beyond code into design/prototyping.

6. **Local/Privacy-First** — ovo-local-llm reflects growing demand for on-device, private agent deployment.
